# CLAUDE.md

视频编辑器项目 - 行为准则

## 项目概述

基于 FFmpeg + PyQt5 的桌面视频编辑工具。

## 功能需求

1. **视频合并** - 多段视频合成一个
2. **音视频混合** - 音乐文件与视频文件合成
3. **字幕添加** - 支持 ASS 和 SRT 格式
4. **文字水印** - 字体大小、颜色、位置、不透明度
5. **输出格式** - MP4(H.264) 和 MKV

## 外部依赖

- FFmpeg: `ffmpeg-8.1.1-essentials_build/bin/ffmpeg.exe`
- Python: PyQt5

## 开发准则

1. **不贸然写代码** - 先想清楚 FFmpeg 命令参数再动手
2. **验证外部依赖** - FFmpeg 必须验证可用
3. **证据先行** - 功能完成后必须实际运行验证
